## Project Name

Asset Verse (Client + Server)

## Purpose

This project is designed to manage assets for a company. Employees can view and request
assets, while HR/admin can manage employees, assets and generate reports. It helps in tracking, returning and documenting company assets efficiently.

## Live URL
https://stupendous-gumption-565b2dsmsagor123.netlify.app/

## Key Features

**Employee Dashboard**
-View assigned assets
-Request new assets
-Return returnable assets
-Search and filter assets

**HR/Admin Dashboard**
-Manage employees
-Manage assets (add/update/detele)
-View all requests and approvals
-Generate PDF reports of assigned assets

**Profile Pages**
-Employee profile
-HR profile with role-specific menu

**Authentication**
-Firebase email/password login
-Role-based access control (Employee / HR/Admin)

**Responsive Design**
-TailwindCSS for responsive and modern UI

**PDF Report Generation**
-Download assigned assets with images
